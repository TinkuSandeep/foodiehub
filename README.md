# FoodieHub

A simple SEO-friendly food website built with Next.js 16, React 19 and TypeScript.

## Pages
- Home
- Menu
- About
- Contact

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm start
```
